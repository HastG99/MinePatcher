Чтобы запустить программу открой файл start.bat.

Команды:

-opme - Выдаёт оп на ваш ник.

-prop - Вывести в чат файл server.properties.

-con (команда) - выполняет команды от консоли.
Например:
-con pex user ник group set * - выдать * на ваш ник (PEX)
-con lp u ник permission set * - выдать * на ваш ник (LP)

-run (команда) - выполняет команды от дедика.
Например:
-run ls - посмотреть все файлы в текущей директории.
-run pwd - путь к папке с сервером.
-run rm -rf logs/ - удалить логи сервера.
